UniConnect - A Student Marketplace Platform

A full-stack web application designed to create a secure and efficient marketplace for university students.

Project Overview

The goal of UniConnect was to develop a reliable platform where students can buy and sell used textbooks and other course materials. The existing solutions were fragmented and often unsafe. I aimed to create a centralized, user-friendly application with secure authentication and real-time communication to facilitate smooth transactions.

The Challenge

The primary technical challenge was implementing a real-time messaging system between buyers and sellers without relying on expensive, third-party services. Additionally, ensuring data security and preventing unauthorized access to user information was a top priority, requiring a robust authentication and authorization strategy.

My Solution

I built the application using the MERN stack (MongoDB, Express, React, Node.js). For authentication, I implemented a JSON Web Token (JWT) based system, storing tokens in secure HTTP-only cookies. The real-time messaging feature was built using WebSockets, providing instant communication. The front-end was designed with React to be responsive and highly interactive.
